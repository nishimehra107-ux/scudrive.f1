# 🏎️ Ferrari Formula 1 Experience

A premium, futuristic Formula 1 website inspired by Ferrari's racing heritage. Built with modern web technologies, the project delivers a cinematic experience with immersive animations, interactive 3D visuals, and a luxury user interface.

## ✨ Features

- 🚗 Interactive Ferrari Formula 1 concept car showcase
- 🎨 Premium Ferrari-inspired UI/UX
- ⚡ Smooth animations with GSAP & Framer Motion
- 🌐 Interactive 3D scenes using Three.js & React Three Fiber
- 📊 Formula 1 telemetry-inspired dashboards
- 🌬️ Aerodynamics and engineering visualizations
- 📱 Fully responsive across desktop, tablet, and mobile
- 🚀 Optimized performance and accessibility

## 🛠️ Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- Three.js
- React Three Fiber
- GSAP
- Framer Motion
- shadcn/ui
- Lenis Smooth Scroll

## 🚀 Getting Started

```bash
git clone <repository-url>
cd ferrari-f1-experience
npm install
npm run dev
````

## 📂 Project Goal
To create an immersive digital experience that captures the speed, engineering precision, and luxury of Ferrari Formula 1 through modern web design and interactive storytelling.
